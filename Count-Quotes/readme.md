Description

Ye ek interactive JavaScript project hai jo New Year 2026 countdown, motivational quotes, aur modal popup feature karta hai.
Project me real-time countdown, auto-rotating quotes, aur manual navigation ka use kiya gaya hai.

Features / Tasks:

Countdown Timer

Shows remaining days, hours, minutes, seconds until New Year.

Pause, resume, or restart countdown dynamically.

Celebration message when event starts.

Motivational Quotes Carousel

Preloaded quotes automatically rotate every 4 seconds.

Manual navigation with Previous and Next buttons.

Welcome Modal Popup

Pops up on page load with a greeting message.

Accessible and closable with a button.

Responsive Design

Works on mobile, tablet, and desktop.

Modern card layout with soft shadows and gradients.

How to Use

Download index.html, style.css, and script.js in the same folder.

Open index.html in your browser.

Countdown starts automatically; quotes rotate every 4 seconds.

Use Pause/Start Countdown button to control timer.

Use Previous/Next buttons to manually navigate quotes.

Close the modal popup when it appears on page load.
